# pfSense Lab

## Objectif
Mise en place d’une infrastructure réseau sécurisée avec pfSense dans un environnement virtualisé.

## Environnement
- VMware
- 2 pare-feux pfSense
- 1 serveur Windows Server
- Réseau LAN / WAN

## Fonctions mises en place
- Configuration des interfaces réseau
- Règles de pare-feu
- NAT
- Accès Internet sécurisé
- Tests de connectivité

## Compétences développées
- Sécurité réseau
- Administration firewall
- Analyse et résolution d’incidents
