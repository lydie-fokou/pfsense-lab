# Architecture du lab pfSense

## Description
Le lab est composé de deux pare-feux pfSense permettant de sécuriser et segmenter le réseau.

## Composants
- pfSense 1 : passerelle principale
- pfSense 2 : pare-feu secondaire
- Windows Server : services réseau
- Machines clientes : tests

## Objectif sécurité
- Filtrage du trafic
- Contrôle des accès
- Protection du réseau interne
